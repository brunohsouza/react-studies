# React Studies

A personal learning project exploring React concepts, built with Vite.

## Tech Stack

- [React 19](https://react.dev/)
- [React Router v7](https://reactrouter.com/)
- [Vite](https://vite.dev/)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Available Scripts

| Script | Description |
|--------|-------------|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build |
| `npm run lint` | Run ESLint |

## Project Structure

```
src/
├── App.jsx          # Root component with routes
├── forms/           # Controlled form examples (text, textarea, select, checkbox, radio)
└── portals/         # React Portals examples
```

## Topics Covered

- **Forms** — Controlled inputs, textarea, select, checkboxes, and radio buttons with unified `handleChange`
- **Portals** — Rendering components outside the root DOM node
